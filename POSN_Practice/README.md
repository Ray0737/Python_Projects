# 🏥 POSN Algorithmic Practice

A collection of competitive programming solutions for **POSN** (The Promotion of Academic Olympiad and Development of Science Education Foundation). These scripts solve various mathematical and logical challenges using Python.

## 📂 Repository Structure

This repository contains 20 distinct solutions to common algorithmic problems (indexed `a1001` through `a1020`).

| File | Type | Challenge Category |
| :--- | :--- | :--- |
| `a1001.py` | Implementation | Basic Arithmetic & Logic |
| `a1002.py` | Implementation | String Manipulation |
| `a1003.py` | Logic | Integer Operations |
| `a1004.py` | Implementation | Conditional Steer |
| `a1005.py` | Data Structures | Lists & Arrays |
| `...` | ... | ... |
| `a1020.py` | Advanced | Complex Problem Solving |

## 🛠️ Tech Stack

- **Language:** Python 3.x
- **Focus:** Performance, time complexity, and memory efficiency for competitive coding.

## 🚀 How to Run

Each file is a standalone script that accepts input from `stdin` (as per standard competitive programming formats).

```bash
# Example: Run problem a1001
python a1001.py
# Then enter the required input values
```

## 📝 Learning Objectives

1.  **Optimization:** Writing code that passes within strict time limits.
2.  **Edge Cases:** Handling large inputs, negative numbers, and empty strings.
3.  **Standard I/O:** Mastering `sys.stdin.read()` for high-volume input processing.

## ⚠️ Notes

Some files (like `a1019.py` and `a1020.py`) are currently placeholders or in-progress solutions. Verify the logic before using them as references.
